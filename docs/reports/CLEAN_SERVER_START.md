## ⚠️ 如果遇到問題

### 問題：端口被占用
```powershell
# 查找占用端口的進程
netstat -ano | findstr :3000

# 停止進程（替換 PID）
Stop-Process -Id <PID> -Force
```

### 問題：內存不足
- 當前配置：4GB
- 可以增加到 6GB 或 8GB（如果系統允許）

### 問題：編譯錯誤
- 查看終端輸出的錯誤訊息
- 檢查 `.env` 文件是否存在
- 確認所有依賴都已安裝
