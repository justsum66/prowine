# 📄 頁面詳細優化建議

## 1. 首頁 (`app/page.tsx`)

### 🎨 視覺設計優化 (15項)

#### 1.1 Hero Section - 沉浸式視覺敘事
**現狀**: 現有輪播較為傳統  
**優化**: 
- 添加全屏視頻背景，展示葡萄園、釀酒過程，營造沉浸式體驗
- 實現視差滾動效果（parallax scrolling），背景以較慢速度移動
- 使用漸變遮罩（gradient overlay）增強文字可讀性：`bg-gradient-to-b from-black/60 via-black/40 to-transparent`
- 添加動態粒子效果，模擬葡萄酒在杯中的流動
- 實現微妙的景深模糊（bokeh effect）突出前景內容

**代碼示例**:
```tsx
// 添加視差滾動視頻背景
<motion.div 
  style={{ y: heroY }} // 視差效果
  className="absolute inset-0"
>
  <video 
    autoPlay 
    loop 
    muted 
    playsInline
    className="w-full h-full object-cover"
  >
    <source src="/videos/wine-heritage-hero.mp4" type="video/mp4" />
  </video>
  <div className="absolute inset-0 bg-gradient-to-b from-black/60 via-black/40 to-transparent" />
</motion.div>
```

#### 1.2 有機形狀設計元素
**優化**: 
- 替換所有直角邊框為柔和圓角（border-radius: 24px）
- 在背景添加有機形狀SVG裝飾（blob shapes）
- 使用不規則的剪裁路徑（clip-path）創造獨特視覺
- 添加流動的漸變背景（fluid gradients）

#### 1.3 Glassmorphism 2.0 效果
**優化**: 
- 為特色卡片添加毛玻璃效果：`backdrop-blur-xl bg-white/10 border border-white/20`
- 實現多層次深度感（layered depth），透過陰影和模糊創造空間感
- 添加微妙的內發光效果（inner glow）

**代碼示例**:
```tsx
<div className="backdrop-blur-xl bg-white/10 dark:bg-black/20 border border-white/20 dark:border-white/10 rounded-3xl shadow-2xl">
  {/* Glassmorphism card content */}
</div>
```

#### 1.4 表達性字體設計
**優化**: 
- 增大Hero標題字體尺寸至 `text-8xl md:text-9xl`，使用粗體serif字體
- 添加動態字體動畫（typing effect）逐字顯示品牌故事
- 使用可變字體（variable fonts）根據滾動位置調整字重
- 實現文字漸變填充效果：`bg-clip-text text-transparent bg-gradient-to-r from-primary-600 to-primary-400`

#### 1.5 3D卡片效果
**優化**: 
- 為WineCard和WineryCard添加3D傾斜效果（perspective transform）
- 實現滑鼠懸停時的3D旋轉：`transform-style: preserve-3d`
- 添加微妙的陰影和光澤效果模擬真實卡片

**代碼示例**:
```tsx
<motion.div
  whileHover={{ 
    rotateY: 5, 
    rotateX: -5,
    scale: 1.02,
    transition: { duration: 0.3 }
  }}
  style={{ transformStyle: "preserve-3d" }}
  className="perspective-1000"
>
  {/* 3D Card */}
</motion.div>
```

#### 1.6 Bento Grid 佈局
**優化**: 
- 將特色區塊重組為Bento Grid佈局（不規則網格）
- 使用不同大小的卡片創造視覺節奏
- 實現響應式網格，移動端自動調整為單列

#### 1.7 動態漸變背景
**優化**: 
- 添加動態漸變背景，顏色根據時間變化（morning/evening tones）
- 使用CSS動畫實現漸變移動效果
- 應用品牌色彩系統的漸變組合

#### 1.8 微妙的動畫裝飾
**優化**: 
- 添加漂浮的幾何圖形作為背景裝飾
- 實現無限循環的流動線條動畫
- 使用SVG路徑動畫創造有機流動感

#### 1.9 圖片品質與優化
**優化**: 
- 使用Next.js Image組件的priority屬性優化首屏圖片
- 實現模糊佔位符（blur placeholder）提升感知性能
- 使用WebP格式並提供AVIF備選
- 添加圖片懶加載（lazy loading）與視口檢測

#### 1.10 品牌視覺一致性
**優化**: 
- 統一所有卡片的間距系統（spacing scale: 4px基數）
- 應用一致的陰影系統（shadow scale）
- 確保所有顏色遵循品牌色板（color palette）

#### 1.11 深色模式優化
**優化**: 
- 為深色模式設計專屬配色方案
- 調整對比度確保可讀性（WCAG AA標準）
- 使用暖色調（warm tones）營造舒適感

#### 1.12 留白藝術
**優化**: 
- 增加區塊之間的留白至 `py-24 md:py-32`
- 使用不對稱留白創造視覺動感
- 實現呼吸感（breathing room）避免擁擠

#### 1.13 視覺層次優化
**優化**: 
- 使用字體大小層級（type scale）建立清晰層次
- 通過顏色對比突出重要元素
- 使用z-index和定位建立空間層次

#### 1.14 圖標系統升級
**優化**: 
- 統一所有圖標尺寸和樣式
- 使用細線風格圖標（thin stroke）符合精品定位
- 添加微妙的圖標動畫效果

#### 1.15 響應式圖片策略
**優化**: 
- 實現藝術指導（art direction），不同設備顯示不同裁剪
- 使用srcset提供多種解析度
- 優化圖片載入策略，優先載入首屏內容

### ⚡ 性能優化 (10項)

#### 3.1 代碼分割優化
**優化**: 
- 使用動態導入（dynamic imports）延遲載入非關鍵組件
- 實現路由級代碼分割（route-based code splitting）
- 分離第三方庫到獨立chunk

#### 3.2 圖片優化策略
**優化**: 
- 使用Next.js Image組件的所有優化功能
- 實現響應式圖片（responsive images）
- 使用CDN加速圖片載入

#### 3.3 字體優化
**優化**: 
- 使用`font-display: swap`避免FOIT
- 實現字體子集化（font subsetting）
- 預載入關鍵字體（preload critical fonts）

#### 3.4 JavaScript優化
**優化**: 
- 移除未使用的代碼（tree shaking）
- 壓縮和混淆JavaScript
- 使用Web Workers處理重計算

#### 3.5 CSS優化
**優化**: 
- 使用Tailwind的purge功能移除未使用樣式
- 實現關鍵CSS內聯（critical CSS inlining）
- 延遲載入非關鍵CSS

#### 3.6 資源優先級
**優化**: 
- 使用`rel="preload"`預載入關鍵資源
- 使用`rel="prefetch"`預取可能需要的資源
- 使用`rel="dns-prefetch"`預解析DNS

#### 3.7 緩存策略
**優化**: 
- 實現Service Worker緩存策略
- 使用HTTP緩存頭（cache headers）
- 實現增量靜態再生（ISR）

#### 3.8 服務端渲染優化
**優化**: 
- 使用Next.js SSG/SSR優化首屏載入
- 實現流式渲染（streaming SSR）
- 優化API響應時間

#### 3.9 第三方腳本優化
**優化**: 
- 延遲載入第三方腳本（analytics, ads）
- 使用async/defer屬性
- 考慮使用Partytown卸載第三方腳本到Web Worker

#### 3.10 監控與分析
**優化**: 
- 實現Web Vitals監控
- 使用Performance API追蹤指標
- 設置錯誤追蹤（error tracking）

### ♿ 無障礙設計 (10項)

#### 4.1 鍵盤導航
**優化**: 
- 確保所有交互元素可通過鍵盤訪問
- 實現邏輯的Tab順序（tab order）
- 添加跳轉鏈接（skip links）

#### 4.2 屏幕閱讀器支持
**優化**:

#### 4.4 焦點指示器
**優化**:

- 確保焦點順序符合邏輯

#### 4.5 表單無障礙

- 提供錯誤訊息的ARIA屬性

#### 4.6 圖片替代文本

- 提供暫停/播放動畫選項

#### 4.8 語言聲明
**優化**: 
- 使用正確的lang屬性
- 標記語言變更內容
- 支持多語言切換

#### 4.9 響應式設計
**優化**: 
- 確保所有功能在移動端可用
- 實現觸摸友好的目標大小（44x44px最小）
- 測試各種屏幕尺寸

#### 4.10 內容結構
**優化**: 
- 使用正確的標題層級（h1-h6）
- 實現清晰的內容層次
- 使用地標元素（landmarks）

### ✍️ 文案優化 (10項)

5. **"不只是葡萄酒，更是一段穿越時空的風土旅程"** - 情感化表達
