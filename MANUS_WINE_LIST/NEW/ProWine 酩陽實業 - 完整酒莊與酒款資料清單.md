| 欄位名稱              | 說明                                                   | 範例                                                                                     |

| `winery`              | 酒莊名稱                                               | `Darioush Estate`                                                                        |
| `region`              | 法定產區                                               | `Napa Valley, Mt. Veeder`                                                                |
| `wine_type`           | 酒款類型                                               | `紅酒 Red wine`                                                                          |
| `grape_varieties`     | 葡萄品種與比例                                         | `83% Cabernet Sauvignon, 9% Cabernet Franc, 5% Merlot, 3% Petit Verdot`                 |
| `vintage`             | 年份                                                   | `2021`                                                                                   |

| `price`               | 品酩價                                                 | `13500元`                                                                                |
| `image_url`           | 酒款照片 URL（ProWine 網站）                           | `http://prowine.com.tw/wp-content/uploads/2021-Darius-II-1.jpg`                          |
| `prowine_url`         | ProWine 酒款頁面 URL                                   | `http://prowine.com.tw/?wine=darioush-darius-ii-2021`                                    |
| `ratings`             | 各專業評分（JSON 物件）                                | `{"decanter": "95", "james_suckling": "93"}`                                             |
| `tasting_notes`       | 品酒筆記                                               | `本款酒酒體厚重又平衡。具備深色莓果、咖啡、比利時巧克力的香氣...`                        |

| `winemaker`           | 釀酒師                                                 | `HOPE GOLDIE，由Michel Rolland提供諮詢服務`                                              |

| `serving_temperature` | 適飲溫度（選填）                                       | `攝氏15-16度`                                                                            |
