# Awwwards 得獎酒莊網站 vs ProWine 逐元素對比分析

## 評分標準說明

Awwwards 評分體系：
- Design（設計）: 40%
- Usability（易用性）: 30%
- Creativity（創意）: 20%
- Content（內容）: 10%

Site of the Month 需達 7.5+/10
Site of the Year 需達 8.5+/10

---

## 標竿案例 1: Pasqua Wines（SOTM 2023年6月）

**得分**: Design 7.87 / Usability 7.46 / Creativity 7.99 / Content 7.98

### 逐元素對比

| 元素 | Pasqua Wines | ProWine 現狀 | 差距分析 | 優化方案 |
|------|-------------|--------------|----------|----------|
| **首頁入口** | WebGL 3D 虛擬酒窖場景，XVIII世紀別墅插圖開場 | 靜態圖片輪播 | 無沉浸感，無品牌記憶點 | 製作品牌形象影片或微動畫 |
| **色彩系統** | #6E6548（橄欖棕）+ #C1380F（磚紅）僅2色 | 5-7色混用，無統一規範 | 色彩雜亂，無品牌識別 | 嚴格限定3色：酒紅+象牙+金 |
| **字體配對** | Acid Grotesk + Wulkan Display | 無明確字體層級 | 缺乏奢華感 | Cormorant Garamond + Noto Sans TC |
| **導航結構** | 3大區塊（品牌/產品/體驗）清晰分類 | 多層級混亂導航 | 用戶迷路 | 扁平化至2層 |
| **動效設計** | 平滑鏡頭移動、3D房間轉場、蝴蝶粒子效果 | 基礎淡入淡出 | 無驚艷感 | 加入視差滾動+微交互 |
| **內容策略** | "House of the Unconventional" 品牌敘事 | 產品堆砌 | 無情感連結 | 建立品牌故事線 |
| **圖片品質** | 專業攝影+AI生成壁畫 | 素材品質參差 | 業餘感 | 投資專業攝影 |
| **互動元素** | 點擊揭示、3D空間探索 | 幾乎無互動 | 被動體驗 | 加入互動式產區地圖 |

### Pasqua 核心設計原則
1. **去傳統化**: 打破葡萄酒行業的保守視覺
2. **沉浸式體驗**: 3D空間讓用戶「進入」品牌世界
3. **極簡色彩**: 僅用2色創造強烈品牌記憶
4. **意外元素**: 老虎、蝴蝶等非傳統元素製造驚喜

## 標竿案例 2: Post Familiar Wine（SOTD 2024年1月）

**得分**: Design 7.44 / Usability 6.92 / Creativity 7.31 / Content 7.44

### 逐元素對比

| 元素 | Post Familiar Wine | ProWine 現狀 | 差距分析 | 優化方案 |
|------|-------------------|--------------|----------|----------|
| **色彩系統** | 純黑白 #000/#FFF 極致對比 | 色彩混亂 | 過度裝飾 | 考慮深色模式+白底方案 |
| **排版風格** | 大字重、高對比、Flat Design | 字體大小不一 | 層級不清 | 建立5級字體系統 |
| **電商整合** | Shopify 無縫購買流程 | 購買流程複雜 | 轉換障礙 | 簡化至3步結帳 |
| **產品展示** | 大圖留白、細節放大 | 小圖密集排列 | 產品不突出 | 大圖+充足留白 |
| **濾鏡效果** | 圖片濾鏡統一視覺調性 | 圖片風格不一 | 雜誌感缺失 | 統一圖片後製風格 |
| **導航模式** | 側邊固定導航 | 傳統頂部導航 | 無特色 | 可考慮側邊導航 |
| **移動優化** | 移動端優先設計 | 響應式斷裂 | 移動體驗差 | 重新設計移動端 |
| **微動效** | 滾動觸發、hover擴展 | 動效不一致 | 缺乏回饋 | 統一3種核心動效 |

### Post Familiar 核心設計原則
1. **極致簡約**: 黑白雙色證明less is more
2. **字體即設計**: 用排版創造視覺衝擊
3. **電商優先**: 每個設計決策都為轉換服務
4. **永恆美學**: 不追趨勢，追求經典

## 給 Cursor 的具體執行指令

### 第一階段：設計系統強制統一（必做）

```
1. 替換 tailwind.config.ts 為新版本
2. 創建 design-tokens.ts 並全站引用
3. 全局搜索替換硬編碼色彩值
4. 統一所有字體到雙字體系統
5. 統一所有間距到8px模組化
```

### 第二階段：首頁重構（必做）

```
1. Hero 改為全屏影片/圖片 + 單一品牌宣言
2. 導航簡化至3項
3. 增加留白至120px區塊間距
4. 實施統一卡片組件
5. 添加品牌色光暈效果
```

### 第三階段：動效統一（必做）

```
1. 定義3種核心動效：fadeInUp / scaleIn / slideIn
2. 統一使用luxury easing函數
3. 移除所有不一致動效
4. 添加scroll-trigger動畫
```

### 第四階段：細節打磨（進階）

```
1. 添加微交互（hover狀態、點擊反饋）
2. 實施骨架屏載入
3. 優化圖片載入體驗
4. 添加頁面轉場動畫
```
